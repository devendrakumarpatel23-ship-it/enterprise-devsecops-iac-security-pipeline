# This file documents the robots directives for the E-Commerce Platform

## Public Pages (should be indexed)
# /              - Homepage (index)
# /products      - Product listing (follow)
# /products/:id  - Product details (index)
# /about         - About page (index)
# /contact       - Contact page (index)

## Private Pages (should NOT be indexed)
# /login         - noindex
# /register      - noindex
# /profile       - noindex, nofollow
# /orders        - noindex, nofollow
# /admin         - noindex, nofollow
# /api/*         - noindex, nofollow
# /checkout      - noindex, nofollow

## Other Directives
# X-Robots-Tag: notranslate
# X-Robots-Tag: unavailable_after:[date]
# X-Robots-Tag: max-snippet:[number]
# X-Robots-Tag: max-image-preview:large
# X-Robots-Tag: max-video-preview:[seconds]
