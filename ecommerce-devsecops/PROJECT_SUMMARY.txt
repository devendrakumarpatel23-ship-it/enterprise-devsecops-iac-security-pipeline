🚀 QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

1. Clone the repository:
   git clone <repo-url>
   cd ecommerce-devsecops

2. Setup environment:
   cp backend/.env.example backend/.env
   cp frontend/.env.example frontend/.env

3. Run all services:
   docker-compose up -d

4. Access the application:
   Frontend:   http://localhost:3000
   API:        http://localhost:5000
   SonarQube:  http://localhost:9000 (admin/admin)
   Grafana:    http://localhost:3001 (admin/admin)
   Prometheus: http://localhost:9090

That's it! All services are running and ready to use.

🔐 SECURITY FEATURES
═══════════════════════════════════════════════════════════════════════════════

✅ Authentication
   • JWT token-based auth with 1-hour expiration
   • Refresh token mechanism (30-day expiration)
   • Password hashing with bcrypt (12 rounds)
   • Session management with Redis

✅ Authorization
   • Role-based access control (RBAC)
   • 3 roles: user, admin, super_admin
   • Permission-based endpoint protection
   • Resource-level authorization

✅ Data Protection
   • Encryption at rest (AES-256)
   • Encryption in transit (TLS 1.3)
   • Field-level encryption for PII
   • Secure secrets management

✅ Network Security
   • VPC with security groups
   • Network ACLs
   • Rate limiting (200/day, 50/hour)
   • DDoS protection ready

✅ Vulnerability Management
   • SAST: Semgrep + SonarQube
   • SCA: npm audit, pip audit, Snyk
   • Container: Trivy image scanning
   • IaC: Checkov validation
   • DAST: OWASP ZAP automation
   • Secrets: GitLeaks detection

✅ Compliance
   • PCI-DSS 3.2.1 (payment processing)
   • OWASP Top 10 2021 (web security)
   • CIS Benchmarks (infrastructure)
   • NIST Cybersecurity Framework
   • SOC 2 Type II ready
   • GDPR compliant
